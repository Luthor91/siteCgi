FAQ - Foire aux questions

Recherchez une question :

Qu'est-ce que CGI ?

CGI (Common Gateway Interface) est une interface de programmation qui permet a des applications web de communiquer avec des serveurs HTTP pour generer des pages web dynamiques.

Quels sont les avantages de CGI par rapport a d'autres technologies web ?

CGI offre une grande flexibilite et une grande liberte pour la conception de pages web, permettant une personnalisation complete des pages en fonction des besoins specifiques du projet. Il est egalement compatible avec de nombreux langages de programmation et de bases de donnees, offrant ainsi une grande compatibilite avec les differentes technologies web.

Comment puis-je me connecter a mon compte utilisateur ?

Pour vous connecter a votre compte utilisateur, vous devez acceder a la page de connexion et saisir votre nom d'utilisateur et votre mot de passe. Si vous avez oublie votre mot de passe, vous pouvez cliquer sur le lien "Mot de passe oublie" pour reinitialiser votre mot de passe.

Comment puis-je modifier mes informations de profil ?

Pour modifier vos informations de profil, vous devez acceder a votre page de profil et cliquer sur le bouton "Modifier". Vous pouvez ensuite saisir les informations que vous souhaitez modifier et cliquer sur le bouton "Enregistrer" pour sauvegarder les modifications.

Comment puis-je contacter l'assistance technique ?

Pour contacter l'assistance technique, vous pouvez utiliser le formulaire de contact disponible sur la page "Contact" de notre site web. Vous pouvez egalement envoyer un e-mail a notre adresse d'assistance technique, qui est indiquee sur la page "Contact".
